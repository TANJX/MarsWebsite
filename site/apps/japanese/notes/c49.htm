    一类形容词

    [ 形 1] 做谓语

    肯定：名＋は＋一类形容词＋です。

     今日は　暑いです。

    否定：词尾“い”变为“く”＋ないです／ありません。

     今日は　暑く　ないです／暑く　ありません。

    过去肯定：词尾 “ い ” 变为 “ かった ” ＋です。

     昨日は　暑かったです。

    过去否定： “ い ” 变为 “ く ” ＋なかったです／ありませんでした。

     昨日は　暑く　なかったです／暑く　ありませんでした。

     

    [ 形 1] 做定语 [ 形 1]+ 名

    広い国

     

    いい [ 形 1]

    现在肯定：いいです

    现在否定：よく　ないです／よく　ありません　

    过去肯定：よかったです

    过去否定：よくなかったです／よく　ありませんでした

    “ いい ” ＋名词

     

    二类形容词

    [ 形 2] 做谓语

    肯定：名词＋は＋二类形容词＋です。

    否定：名词＋は＋二类形容词＋では／じゃ　ありません。

    过去肯定：名词＋は＋二类形容词＋でした。

    过去否定：名词＋は＋二类形容词＋では／じゃ　ありませんでした。

     

    [ 形 2 ] 做定语 [ 形 2 ] ＋な + 名

     李さんは　きれいな　人です。

     

    表示感情

    [ 名 1 ] は　 [ 名 2 ] が　 + 　 [ 一类形／二类形 ] です。

     わたしは　犬が　怖いです。

     

    表示能力

    [ 名 1 ] は　 [ 名 2 ] が　 + 　 [ 一类形／二类形 ] です。

    “ 分かります、できます、上手です、下手です、苦手です ” 等々。

     小野さんは　料理が　上手です。

     

    比较句

    二者相较

    [ 名 A] は　 [ 名 B] より　 [ 形容词 ] です。

    A>B

    “~ （名 A ）比 ~ （名 B ）更 ……”

     東京は　奈良より　にぎやかです。

    [ 名 A] より　 [ 名 B] の　ほうが　 [ 形容词 ] です。

    A<B

    “比起~（名A）来，~（名B）更……”

    [ 名 A] は　 [ 名 B] ほど　 [ 形 1 （い → く） ] ないです／ [ 形 2] では　ありません。

    A<B

    “ 名 A 不如名 B 那么 ……”

    [ 名 A] と　 [ 名 B] と　どちらが　 [ 形容词 ] ですか。

    “ 名 A 和名 B ，哪一个更 …… ？ ”

     日本語と　英語と　どちらが　難しいですか。

    —— 日本語／英語の　ほうが　難しいです。

    ——どちらも　難しいです。

     

    三者以上

    [ 名 1] の　中で　 [ 名 2] が　いちばん　 [ 形容词 ] です。

    “ 在 ~ （名 1 ）之中， ~ （名 2 ）最 ……”

     スポーツの　中で　サッカーが　いちばん　おもしろいです。

    名 1 本身具有范围的概念，则不需要加 “ の中 ” 。

    [ どれ／どの + 名／何／いつ／どこ／だれが ] 　いちばん　 [ 形容词 ] ですか。

      “ 哪个 ~ 最 …… ？ ”

     家族の　中で　だれが　一番　背が　高いですか。

    当上下文中明确列举出了在哪几个事物中比较，疑问词则可以用

    “どれ／どの+名”，如没有则用“何”。

     りんごと　ナシと　バナナの　中で、どれが　いちばん　好きですか。

     果物の　中で　何が　いちばん　好きですか。

    助词

    [ 名 ] が

    表示情感或能力的对象。

    [ 名 ] や [ 名 ]

    表示不完全列举

    [ 名 ] など

    置于列举事物的最后一项后面

    ～から [ 接续助词 ]

    [ 表示原因、理由的小句 ] ＋から

    だから、 …[ 连词 ]

    だから、 [ 表示结果、结论的小句 ] 。

    “ ですから ” 为 “ だから ” 更加礼貌的说法。

     

    连词

    そして：表示并列关系， “ 而且 ” 。

    でも：表示转折关系， “ 但是 ” 。

     わたしの　部屋は　狭いです。そして、汚いです。

     

    疑问连体词

     どんな＋名词

    询问某人或某事物的性质， “ 什么样的 …… ？ ”

     あの　人は　どんな　人ですか。

    —— おもしろい人です。

     

     

    惯用表达

    どうですか。

    询问对方对某事物的看法或感受， “ 如何？怎样？ ”

    この　料理は　どうですか。 [ 正在吃的时候 ]

    京都は　 どうでしたか 。 [ 对方去京都旅游归来之后 ]

    お茶、どうですか。 [ 推荐或询问建议 ]

    いかがですか。 [ 询问尊长 ]

     

    どうしてですか。

    询问事由， “ 为什么 ” 。

    回答时在谓语后面加 “ から ” 。
