动词て形

 

 

 

用法

[ 动 ] て　 [ 动 ]

用于连接前后两个动词 / 句节

两个动作存在相继发生，并列，弱因果的关系，或前者为后者的手段、方法、方式，附带状况等

[ 动 ] てから　 [ 动 ]

相继发生， “…… 之后 ……”

A 、 B 先后顺序不可逆。

[ 动词 ] て＋ください

请求对方做某事

[ 动词 ] て + います。

（前接持续性动作时）表示动作或变化正在进行

（前接瞬间性动作或变化动词时）表示结果状态

[ 动词 ] ても + いいです

表示许可， “ 可以 ……”

[ 动词 ] ては＋いけません

表示禁止， “ 不可以 ……”

[ 动词 ] ても + いいです＋か

向对方征求许可， “ 我可以 …… 吗？ ”

 

 

形 1 、形 2 、名词 て 型

形 1 的て形：词尾い→くて

形 2 的て形：词干＋で

名词的并列形式： [ 名 1] ＋で +[ 名 2]

 

 

数量词

不同种类名词与量词的搭配组合

名 [ 时间 ] ＋ [ 动 ]

表示动作或状态的持续时间

名 [ 时间 ] ＋に＋名 [ 次数／量 ] ＋ [ 动 ]

表示频率或量

 

助词

[ 动 1 ，动 2 （ます连用形）／ サ变动词词干 ] ＋に＋行きます／来ます。

表示移动行为的目的。

名 [ 数量 ] ＋で

表示总计

名 [ 场所 ] ＋を＋ [ （表示移动、经过或离开意义的）动词 ]

表示经过或离开某场所

通ります、渡ります、過ぎます

名 [ 附着点 ] ＋に＋ [ 动 ]

表示人或物体的附着点，动作的着落点

が

接续助词

连接两个小句，表示转折
